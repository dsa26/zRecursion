# zRecursion
Problems in this assignment:
1. Factorial
2. Fibonacci
3. Sum of digits
    * Calculates the sum of all the digits in a number
4. Towers of Hanoi
5. Subset sum
    * Calculates the number of ways in which a target sum can be achieved using the numbers in an input list
